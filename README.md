# 🔐 StringGuard

**StringGuard** is a secure, minimal, and privacy-focused text storage application inspired by the core functionality of ProtectedText.

Built with a modern stack using **React** and **Supabase**, StringGuard allows users to create and access protected text entries without requiring traditional accounts—keeping the experience fast, simple, and anonymous.

---

## ✨ Features

- 🔑 **Password-protected text pages**
- ⚡ **Instant access with no sign-up required**
- ☁️ **Cloud-backed storage with Supabase**
- 💻 **Responsive and clean user interface**

---

## 🧱 Tech Stack

- **Frontend:** React
- **Backend / Database:** Supabase
- **Authentication & Security:** Supabase Auth / Custom logic

---

## 🎯 Project Goal

The goal of StringGuard is to replicate and improve upon the simplicity of ProtectedText while leveraging modern web technologies for better performance, scalability, and developer experience.

---

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/stringguard.git

# Navigate into the project
cd stringguard

# Install dependencies
npm install

# Start the development server
npm run dev
```
